## Instructions
- 移動(Move)：矢印キー(Arrow keys)
- 選択(Select)：Spaceキー(Space key)
- メニュー表示(Show menu)：mキー(Press m key)
- メニュー非表示(Hide menu)：bキー(Press b key)
