KU_RPG  
===
とある香川の大学を舞台としたRPGゲームのチーム開発  
## 操作
- 移動：矢印キー
- 選択：Spaceキー
- メニュー表示：mキー
- メニュー非表示：bキー
## 注意
**クラシック版につき、以下の要素が含まれます。**
- 多少のゲームバランス崩壊
- 予期せぬ不具合の可能性
